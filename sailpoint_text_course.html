what is an identity?

In simple words, Idenitity is a user profile in an organistation which tells who you are and what you do in the organistation. And this sailpoint IdentityIQ as a software manages these identites.

How does sailpoint work?

If we dig any organistation all you find is applications and there are users[identites] who got access or permissions to use this applications, Here managing this huge amount of applications mannuly is nearly impossible so usually every company use IDM.
Sailpoint Onboarding all these applications do all the work in central platform.


If you see the image sailpoint has two important Modules Compliance manager and Lifecycle manager. Compliance module is all about implimenting risk, policy and performing certifications on the other hand lifecyle manager revolves around provisioning part and lifecycle events. How to install Sailpoint IdentityIQ

Can't read whole thing? Then watch how to install sailpoint

Installing sailpoint is a long process but easy to do, If you want to see sailpoint in your pc there are 4 componets you should download
1) Java: Sailpoint Identity IQ runs of java api. How to download java for sailpoint
2) Database: Everything in sailpoint is an object so we need database to store this object. MySql Installation For Sailpoint
3) Application Server: Sailpoint is a web based application so we need application server to host sailpoint in web. Tomcat Installation
4) Sailpoint software: Finally sailpoint software. SailPoint Installation

Java Installation For SailPoint

Ok lets start installing with java. Downloading java is pretty stright forward go to google and type "Download Java" or use this link to download java


- Click on download java.

- You will see an exe file downloding down left on your screen

- Double click on the downloaded .exe file. It asks for changes on your computer. Click on the Yes button to run the setup.

In the installation wizard window, click on the Next button.
Click on the Change button if you want to change the installation directory. By default, it installs JDK on the location C:\Program Files\Java\jdk-14.0.1. Otherwise, click on the Next button. It starts the installation of JDK.

When Java is successfully installed on your system it shows a message Java SE Developments Kit 14.0.1 (64-bit) Successfully Installed. Click on the Close button.

Then we have to set up Environment Varibles, In search tab type Environment variables and open it.

Click on Environment Variables, In user variables for human click "New" and variable name type : "JAVA_HOME" and in variable path : "C:\Program Files\Java\jdk-18.0.1.1" and click ok.

Then go to system variables click on path and click on new and paste this path : "C:\Program Files\Java\jdk-18.0.1.1\bin" click ok, ok and ok.

We have installed Java successfully. To check the Java is successfully installed or not, follow the steps given below.
Open the Command Prompt and type the following commands:
Type the command java ?version to check the Java version. It ensures that Java is successfully installed in your system.

If you can see the java version displayed in command prompt then you are all set, but if find this error "java not is not recognized" then you made some mistake while setting your Environment variables.

MySQl Installation For Sailpoint

Now we are in our secound step, we need a database we will use mysql database but you install any database sailpoint support 4 different types of datatypes Mysql,sql,oracle and db2. Most of the time people use MySql so let's start installing MySql For sailpoint.

Step 1: Go to the official website of MySQL and download the community server edition software. Here, you will see the option to choose the Operating System, such as Windows.
Step 2: Go to Archives and choose version 5.7.30

Step 3: Next, there are two options available to download the setup. Choose the version number for the MySQL community server, which you want. If you have good internet connectivity, then choose the mysql-installer-web-community. Otherwise, choose the other one.


Step 1: After downloading the setup, unzip it anywhere and double click the MSI installer .exe file. It will give the following screen:

Step 2: In the next wizard, choose the Setup Type. There are several types available, and you need to choose the appropriate option to install MySQL product and features. Here, we are going to select the "server only"option and click on the Next button.


- And dont change anything follow the next next finish process.

- Note: Please don't forget your username and password.

- Then click on finish. if you have any doubt watch Installation Video.

After MySql Installation we have to set Environment Variables for mysql. Open Environment variables, in system variables click on path then click on new then paste mysql path up to bin 'C:\Program Files\MySQL\MySQL Server 5.7\bin' click ok, ok and ok.

Check if MySQl is installed properly or not.

Open command prompt and type "mysql -u root -p" and enter then it will ask your password type it and click on enter.

If see this then you are my sql installation is done. If you face this error "mysql not recognized" then you made some mistake while doing enviornment variables.
